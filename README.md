# Smart-footfall-Analysis-system-for-retail-optimization
Smart footfall Analysis system for retail optimization powered by computer vision and Stream Processing

YOLO3 Weights file: https://github.com/patrick013/Object-Detection---Yolov3/blob/master/model/yolov3.weights

## How to run the application
1. Download the yolo3.weights file from the above link and place it in the directory
2. Activate the Python environment
3. Run the app.py file
4. It will open the UI, and from there can run the application
5. First, start the system from the Operator UI before starting the footfall analysis
6. Via the UI can access the Power BI dashboard and the Grafana Dashboard
